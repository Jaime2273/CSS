Qué es CSS y para qué sirve

CSS es un lenguaje utilizado para describir la presentación visual de las páginas web y sirve para controlar el diseño y el estilo de un documento HTML, especificando aspectos como colores, fuentes, márgenes, espaciados, alineaciones, fondos y otros elementos de diseño visual.
