Ejemplo de Cuadrado con CSS

Este cuadrado tiene las siguientes propiedades de CSS aplicadas:

  • Width (Ancho): El cuadrado tiene un ancho de 200 píxeles.
  • Height (Altura): La altura es de 200 píxeles.
  • Padding (Relleno): Se añaden 20 píxeles de espacio interno entre el contenido y el borde del cuadrado.
  • Border (Borde): El borde es sólido, negro, y tiene un grosor de 5 píxeles.
  • Margin (Margen): El cuadrado está separado de otros elementos con un margen externo de 30 píxeles.
