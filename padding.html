Modelo de Caja en CSS

Este ejemplo muestra la estructura del modelo de caja en CSS:

  • Marco Externo: Tiene un borde gris de 10px, con un padding (espacio interno) de 20px.
  • Marco Interno: Tiene un borde azul de 10px y un padding adicional de 20px.
  • Contenido Real: Es el área naranja que representa el contenido en sí, sin márgenes o bordes adicionales.
