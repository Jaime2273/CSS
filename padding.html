Cuadro con Margen, Padding y Border

Este ejemplo muestra cómo se usan las propiedades de CSS:

  • Margin (Margen): Define el espacio externo alrededor de un elemento, separando dicho elemento de otros elementos cercanos.
  • CSS Padding (Relleno o Espaciado Interno): Define el espacio interno entre el contenido del elemento y su borde. Es decir, el espacio dentro del contorno del elemento, alrededor de su contenido.
  • CSS Border (Borde): Define un contorno visible alrededor de un elemento. Puede tener diferentes grosores, estilos y colores.
  • CSS Width and Height (Ancho y Altura):Estas propiedades definen el tamaño de un elemento, es decir, su ancho (width) y su altura (height).
  1. Width/Height establece el tamaño básico del elemento.
  2. Padding añade espacio interno adicional dentro del borde del elemento.
  3. Border crea una línea visual alrededor del elemento.
  4. Margin añade espacio externo entre el borde del elemento y otros elementos
